#### [problem](https://typehero.dev/challenge/first-of-array) 

Implement a generic First<T> that takes an Array T and returns its first element's type.

For example:

```typescript
type arr1 = ['a', 'b', 'c']
type arr2 = [3, 2, 1]

type head1 = First<arr1> // expected to be 'a'
type head2 = First<arr2> // expected to be 3
```

#### challenge
```typescript
type First<T> = unknown
```

#### test 

```typescript
import type { Equal, Expect } from '@type-challenges/utils'

type cases = [
  Expect<Equal<First<[3, 2, 1]>, 3>>,
  Expect<Equal<First<[() => 123, { a: string }]>, () => 123>>,
  Expect<Equal<First<[]>, never>>,
  Expect<Equal<First<[undefined]>, undefined>>,
]

type errors = [
  // @ts-expect-error
  First<'notArray'>,
  // @ts-expect-error
  First<{ 0: 'arrayLike' }>,
]
```

#### my answer

First 문제는 말그대로 배열에서 첫 번째 인자를 추출하는 타입을 만드는 문제. 총 3가지 버전으로 풀 수 있다.

```typescript
type First<T extends any[]> = T extends [] ? never : T[0]
```
```T```가 빈 배열인지 확인하고 빈 배열이 아니라면 0번 인덱스를 반환하는 타입


```typescript
type First<T extends any[]> = T["length"] extends 0 ? never : T[0]
```
마찬가지로 배열 T의 개수를 확인하고 0이 아니라면 0번 인덱스를 반환한다


```typescript
type First<T extends any[]> = T extends [infer F, ...any[]] ? F : never;
```
infer로 추론하는 타입. ```[infer F, ...any[]]``` 여기서 배열이 0이 이상인지 확인하는 동시에 infer로 첫 번째 요소를 추론한다.
